# Grocery Tracker App Frontend

> Backend code can be found [here](https://github.com/Lukec-lehighu/CSE371-Project-Server)

A web app for tracking purchases and debts for houses with several roommates.

Developed for Lehigh University's CSE-371 course.

## Installation:

1. Download the repository using `git clone`
2. Open project folder 
    - `cd CSE371-Project`
3. Install `serve` if you don't already have it
    - `npm install -g serve`
4. Run deploy script
    - `sh deploy.sh`

The frontend will now be served on your local network
